## `core sdk:link TARGETSDKFOLDER`

Link an specific SDK folder given the parameter

```
USAGE
  $ core sdk:link TARGETSDKFOLDER

ARGUMENTS
  TARGETSDKFOLDER  The path to the lisk SDK folder

EXAMPLE
  sdk:link /sdk/location/
```
